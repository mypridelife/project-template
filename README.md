# 前端项目模板

## 技术栈：

- 框架：vue
- 组件库：iview、element、vant

### 管理端

​	project-admin（基于element-admin）

​	组件库基于element

### 手机端

​	project-h5（空白项目）

### 微信小程序

​	prioject-wxapp（基于农产品收集工具0.2）

​	组件库基于iview和vant

## 内容

- 以上所有项目均已区分环境。
	- 日常
	- 预发
	- 正式
- 管理端和手机端加入钉钉免登

## 运行

```
npm i
npm run dev
```

## 构建

```
npm run build:dev/pre/prod
```
